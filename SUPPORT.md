# Support

Use the following resources before opening an issue:

- [README.md](README.md) for installation, reading, editing, and runtime examples.
- [docs/format.md](docs/format.md) for the supported PC t3db format version 8 binary layout.
- [docs/dll-parity.md](docs/dll-parity.md) for verified DLL behavior and intentional differences.

For reproducible bugs, use the bug-report template and include the package version, runtime, operating system, error output, and the smallest safe reproduction. Do not upload proprietary database files; describe the affected table and field or construct a minimal synthetic input instead.

Feature proposals should explain the use case and any expected API, metadata, or binary-format impact. Xbox byte order, archive handling, filesystem helpers, CLI behavior, and game business logic are currently outside the library's scope.

Security vulnerabilities must be reported privately according to [SECURITY.md](SECURITY.md). This project cannot provide support for Electronic Arts accounts, installations, licensing, or game operation.
