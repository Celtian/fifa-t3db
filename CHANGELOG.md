## Project bootstrap

### Added

- Portable PC t3db format version 8 reader and editor.
- Lazy table materialization for all supplied fixture tables.
- Composite-key inference for updates, inserts, and deletions.
- Deterministic Huffman reconstruction and FIFA CRC regeneration.
- Strict TypeScript, ESLint, Vitest, Bun, Husky, and lint-staged tooling.
- DLL parity, binary-format, contribution, security, and support documentation.
- GitHub Actions validation, coverage reporting, Ubuntu package checks, and tag publishing.
- Automated changelog generation from semantic version tags and Git history.
- Bun-based patch, minor, major, and beta release helpers.
- Staged npm and GitHub Packages manifests generated without mutating root metadata.
